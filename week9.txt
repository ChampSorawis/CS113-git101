week 9 structures , unions
- Structure
  เป็นการกำหนดรูปแบบข้อมูลที่มีการรวบรวมตัวแปรข้อมูลหลายเรื่องที่สัมพันธ์กัน อาจเป็นข้อมูลคนละประเภทกันหรือเป็นอาเรย์ก็ได้ ชื่อสมาชิกที่อยู่ใน structure เดียวกันห้ามซ้ำกันได้ไปซ้ำใน structure อื่นได้ ถ้ากำหนดค่าน้อยกว่าจำนวนสมาชิกที่ประกาศไว้ใน structure จะให้ค่าเป็น 0 หรือ NULL
- Structure Declaration
  structure tag จะมีหรือไม่มีก็ได้
- Self Reference
  ทำได้ก็ต่อเมื่อ structure ที่จะเรียกใช้อีกใน structure เป็น pointer
- Storage in Memory
  ไม่สามารถใช้กับ operator == และ != ได้เนื่องจากสมาชิกใน structure มี bytes ที่ใช้เก็บแตกต่างกัน และวิธีการเก็บคำจะเก็บแบบทีละ 2 หรือ 4 bytes การเข้าถึงสมาชิกแต่ละสมาชิกสามารถทำได้โดยใช้ dot operator (.) หรือ pointer operator (->)
- Typedef
 การกำหนดประเภทข้อมูลขึ้นมาเพิ่มเติมโดยกลายเป็นประเภทข้อมูลใหม่ที่ใช้เพียง 1 คำแทนประเภทของข้อมูลนั้น
- Union
  คล้ายกับ structure ตรงที่มีการใช้พื้นที่ทำงานร่วมกันแทนที่จะเสียที่เก็บข้อมูลที่ไม่ได้ใช้ จำนวน byte ที่จะใช้เก็บต้องมีอย่างน้อยที่จะเก็บข้อมูลสมาชิกที่ใช้พื้นที่มากที่สุด
- Union Definition
  จะมีการเริ่มต้นด้วยข้อมูลประเภทเดีวกับประเภทของข้อมูลสมาชิกตัวแรก
- Enumeration หรือ enum
  คือการกำหนดค่าคงที่ให้กับตัวแปรตามลำดับ โดยสามารถตั้งค่าให้กับสมาชิกตัวแปรได้ว่าจะไม่ต้องเริ่มต้นที่ 0
