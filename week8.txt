week 8 linked list
1) linked list 
เป็นโครงสร้างข้อมูลที่เก็บข้อมูลโดยข้อมูลแต่ละตัวเชื่อมโยงกันเป็นสาย เพื่อให้สามารถเข้าถึงข้อมูลแต่ละตัวได้
2) structure
เป็นการกำหนดรูปแบบข้อมูลที่มีการรวบรวมตัวแปรข้อมูลหลยเรื่องที่สัมพันธ์กัน อาจเป็นข้อมูลคนละประเภทกัน
- การประกาศตัวแปรประเภท structure
  การเข้าถึงสมาชิกแต่ละฟิลด์สามารถทำได้โดยใช้ dot operator (.) หรือเรียกว่าการเข้าถึงแบบ dereferencing operator
- ตัวแปร pointer ไปยัง pointer
  นอกจากการใช้ dereferencing operator กับตัวแปร pointer แล้วสามารถใช้ structure pointer operator (->) หรือเรียกว่า arrow operator 
- การใช้ typedef
 การกำหนดประเภทข้อมูลขึ้นมาเพิ่มเติม สามารถใช้ร่วมกับ struct ได้โดยกลายเป็นประเภทข้อมูลใหม่ที่ใช้เพียง 1 คำแทนประเภทของข้อมูลนั้น
- self-referential structures
ใช้จัดการ linked list โดยสร้างขึ้นมาโดยใช้ struct และภายใน struct มีฟิลด์ที่เป็น pointer เพื่อใช้อ้างอิงกลับ มักจะใช้คำว่า node ในการอ้างถึงหน่วยที่เก็บข้อมูล 1 หน่วย ซึ่ง struct node จะเก็บข้อมูล 2 ชนิด คือ
  1. ฟิลด์ data คือส่วนที่ใช้เก็บข้อมูลประเภท int
  2. ฟิลด์ next ใช้เก็บตำแหน่งในหน่วยความจำถัดไป เป็นตัวแปร pointer ที่ใช้อ้างอิงไปโหนดถัดไป
3) การเข้าถึงข้อมูลแต่ละตัวใน linked list (Traversal)
4) การลบข้อมูลออกจาก linked list (deletion)
5) การเพิ่มข้อมูลเข้าไปใน linked list (insertion)
- การเพิ่มข้อมูลเข้าไปในส่วนต้นของ list
- การเพิ่มข้อมูลเข้าไปในตอนกลางของ list
6) doubly linked list
ที่เห็นผ่านมาจะเป็นแบบ singly linked list คือพิจารณาทีละโหนดที่อยู่ติดกันเชื่อมต่อถึงกันผ่านลิงก์เดียว แต่แบบนี้จะเชื่อมกันด้วยลิงก์ 2 ลิงก์ โดยลิงก์อันนึงจะเชื่อมกับโหนดถัดไป ส่วนอีกลิงก์จะเชื่อมต่อกับโหนดก่อนหน้า
7) circular linked list
- singly circular linked list
- doubly circular linked list
