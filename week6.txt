Array คืออะไร 

Array คือ กลุ่มของตัวแปรที่ถูกจัดรวมกัน โดยการเรียกใช้นั้น เราต้องการเรียกใช้ตัวแปรในอาร์เรย์ ตัวแปรแต่ละตัวจะมีเบอร์กํากับ (index) อยู่ เช่นA[1],A[2] ใช้งานเหมือน int ปกติ int b; b = A[2] + 14; 

กําหนดขนาดอาร์เรย์ตามจํานวนค่าตั้งต้น 

int A[] = {0, 1, 2, 3, 4}; 

ขนาดของArrayควรเป็นค่าคงที่(constant)เท่านั้น 

#Define 

Define คือ เป็นคำสั่งที่สั่งให้Complierเปลี่ยนอักษรที่เรากำหนดนั้นเป็นอักษรที่เรากำหนดเช่น 

#define arraySize 5 

int A[arraySize]; 

จะกลายเป็น 

int A[5]; 

Pass By Value 

ก็อปปี้ค่าของตัวแปรจากผู้เรียกไปให้ฟังก์ชัน 

สร้างตัวแปร local variable ใหม่ในฟังก์ชัน 

การเปลี่ยนแปลงค่าในฟังก์ชันไม่ส่งผลถึงตัวแปรต้นตํารับ 

pass by reference 

ส่งตําแแหน่งของตัวแปรจากผู้เรียกไปให้ฟังก์ชัน 

ตัวแปรในฟังก์ชันเป็นตัวเดียวกันกับในผู้เรียก 

การเปลี่ยนแปลงค่าในฟังก์ชันเปลี่ยนตัวแปรต้นตํารับด้วย
