-ประเภทข้อมูลและขนาด (Data Types and Sizes) 

ในภาษาCขนาดของประเภทของข้อมูลมีขนาดไม่เท่ากัน โดยขนาดแต่ละอันสามารถใช้sizeofในการเช็คขนาดและข้อมูลได้ เช่น 

#include <stdio.h> 

    int main() 

    {   printf("sizeof(char)   = %d\n", sizeof(char)); 

        printf("sizeof(int)    = %d\n", sizeof(int)); 

        printf("sizeof(float)  = %d\n", sizeof(float)); 

        printf("sizeof(double) = %d\n", sizeof(double)); 

        printf("sizeof(short)  = %d\n", sizeof(short)); 

        printf("sizeof(long)   = %d\n", sizeof(long)); 

    } 

โดยแต่ละประเภท มีขนาดดังนี้ 

  - char 1 byte 

  - int 4 bytes 

  - float 4 bytes 

  - double 8 bytes 

  - long 4 or 8 bytes 

  - void 1 byte

Printf ไม่ขึ้นบรรทัดใหม่ แต่ putsขึ้นบรรทัดใหม่
